Pilas

Una pila es una estructura de datos donde los elementos se añaden y se quitan según el principio de “último en entrar, primero en salir” (LIFO). Por ejemplo, en una pila de libros, el último libro que se coloca es el primero que se retira. En Java, se puede implementar pilas mediante la clase genérica Stack. (Quispe, 2024)

Métodos más usados.

  • `push`: inserta un elemento en la pila.
  • `pop`: elimina el elemento superior de la pila.
  • `peek`: devuelve el elemento superior de la pila sin eliminarlo.
  • `isEmpty`: verifica si la pila está vacía.
  • `search`: busca un elemento en la pila y devuelve su distancia desde la parte superior.

    import java.util.Stack;

    public class PilaLibros {
        public static void main(String[] args) {
            Stack pilaDeLibros = new Stack<>();
            
            // Agregando libros a la pila
            pilaDeLibros.push("El principito");
            pilaDeLibros.push("Cien años de soledad");
            pilaDeLibros.push("Don Quijote de la Mancha");
            
            // Mostrando el libro en la cima sin eliminarlo
            System.out.println("Libro en la cima: " + pilaDeLibros.peek());
            
            // Sacando libros de la pila
            System.out.println("Sacando: " + pilaDeLibros.pop());
            System.out.println("Sacando: " + pilaDeLibros.pop());
            
            // Verificando si la pila está vacía
            System.out.println("¿La pila está vacía? " + pilaDeLibros.isEmpty());
            
            // Buscando un libro en la pila
            int posicion = pilaDeLibros.search("El principito");
            if (posicion != -1) {
                System.out.println("El libro 'El principito' está en la posición: " + posicion);
            } else {
                System.out.println("El libro no se encuentra en la pila.");
            }
        }
    }
