Colas

Una cola es una estructura de datos donde los elementos se añaden al final y se retiran desde el principio, siguiendo el principio de “primero en entrar, primero en salir” (FIFO). Por ejemplo, en una cola de banco, el primer cliente que llega es el primero en ser atendido, o en un laboratorio, el primero en llegar es el primero que se hace exámenes. En Java, podemos implementar colas utilizando la interfaz Queue. (Quispe, 2024)

Métodos más usados.

  • `add`: inserta un elemento en la cola.
  • `poll`: elimina y devuelve el primer elemento de la cola.
  • `peek`: devuelve el primer elemento de la cola sin eliminarlo.
  • `isEmpty`: verifica si la cola está vacía. 

    import java.util.LinkedList;
    import java.util.Queue;

    public class ColaBanco {
        public static void main(String[] args) {
            Queue colaClientes = new LinkedList<>();
            
            // Agregando clientes a la cola
            colaClientes.add("Cliente 1");
            colaClientes.add("Cliente 2");
            colaClientes.add("Cliente 3");
            
            // Mostrando el primer cliente sin eliminarlo
            System.out.println("Primer cliente en la cola: " + colaClientes.peek());
            
            // Atendiendo clientes (eliminándolos de la cola)
            System.out.println("Atendiendo a: " + colaClientes.poll());
            System.out.println("Atendiendo a: " + colaClientes.poll());
            
            // Verificando si la cola está vacía
            System.out.println("¿La cola está vacía? " + colaClientes.isEmpty());
        }
    }
